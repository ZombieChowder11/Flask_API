# Python Flask RESTful API
This is a Python API utilizing standard CRUD methodologies. It allows user registration to a database, creating, edditing and deleting content. <br>

## Deployment
1. Install Python with Flask
2. Install mySQL
3. run "python app.py"

**Build With**<br>
1. Python<br>
2. Flask<br>
3. mySQL<br>
